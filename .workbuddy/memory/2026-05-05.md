## ProjectHub - 登录页重设计

重新设计 `projecthub/src/views/LoginView.vue`，采用赛博朋克风格 + 玻璃态设计：

### 设计变更
- 移除所有多余文字（RSA加密说明、2小时有效期、自动保存等）
- 背景：深色主题 + 动态网格线 + 浮动霓虹光晕球（青色/紫色/粉色）
- 品牌标识：渐变文字 + 旋转六边形Logo + 光晕效果
- 表单卡片：玻璃态效果（backdrop-filter blur + 半透明边框）
- 输入框：深色半透明背景，focus时霓虹青色光晕
- 提交按钮：渐变背景（青色→紫色），hover时颜色过渡
- 动画：网格线移动、光晕球浮动、Logo旋转、表单淡入

### 配色方案
- 主色：青色 #00F0FF、紫色 #7B61FF、粉色 #FF61D8
- 背景：深蓝黑 #0a0e1a
- 文字：白色 + 半透明白色层次

### 技术要点
- 移除旧的hero区域说明列表
- 使用 `:deep()` 穿透TDesign组件样式
- CSS动画性能优化（transform + opacity）
- 响应式适配移动端（max-width: 480px）

## ProjectHub - 登录页二次调整（浅色系）

根据反馈调整为浅色系互联网风格：

### 设计变更
- **配色调整**：从深色赛博朋克改为浅色系（白色背景、#f5f7fa 渐变背景）
- **隐藏标签**：移除邮箱和密码的 form-item label（`label` 属性删除，`:deep(.t-form__label) { display: none }`）
- **按钮间距**：为 `.submit-btn` 添加 `margin-top: 8px` 增加与输入框的间距
- **自动填充处理**：添加 `-webkit-autofill` 样式，防止浏览器自动填充时背景变黄
- **移除深色元素**：删除背景网格线、光晕球等深色特效
- **Logo配色**：从青色渐变改为蓝色系（#4a90d9 → #7B61FF）

### 技术要点
- 使用 `:deep(.t-input__inner:-webkit-autofill)` 处理浏览器自动填充
- 输入框高度统一为 48px
- 卡片阴影采用多层阴影提升层次感
- placeholder 颜色使用 #a0aec0
