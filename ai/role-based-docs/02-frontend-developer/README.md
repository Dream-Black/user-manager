## 🔧 开发注意事项

### 1. 电脑识别
- 使用 `navigator.userAgent` 或 `location.hostname` 获取电脑标识
- 通过请求头传递给后端

### 2. 本地代理连接
- 默认地址：`http://localhost:6789`
- 图片URL拼接本地代理地址
- 需要处理代理服务未启动的场景

### 3. 图片压缩
- 封面图必须压缩到200KB以下
- 使用 `utils/imageCompress.js` 工具
- 目标分辨率：宽度800px

### 4. 路由守卫
- 进入资源模块前检查本地代理服务状态
- 未配置路径时引导到设置页
