# 页面设计需求

## 📍 页面结构

```
资源管理
├── /resources                    # 漫画列表页
├── /resources/comics/:id        # 漫画阅读页
└── /resources/settings           # 路径设置页
```

---

## 1. 漫画列表页 `/resources`

### 页面功能
- 显示当前电脑名称
- 显示当前资源路径
- 网格展示漫画卡片
- 支持搜索漫画名称
- 支持筛选漫画类型

### 页面布局
```
┌──────────────────────────────────────────────────────────┐
│ [面包屑] 资源管理 / 漫画                                  │
├──────────────────────────────────────────────────────────┤
│                                                          │

│  │ [电脑图标] 我的台式机    [路径图标] D:/MyComics/   │  │
│  │  ⚙️ 设置                    🔄 重新扫描            │  │
│  └────────────────────────────────────────────────────┘  │
│                                                          │
│  [🔍 搜索漫画...]  [类型筛选 ▼]  [排序 ▼]              │
│                                                          │
│  ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐           │
│  │        │ │        │ │        │ │        │           │

│  │        │ │        │ │        │ │        │           │
│  ├────────┤ ├────────┤ ├────────┤ ├────────┤           │
```

### 交互说明

#### 漫画卡片点击
1. 单击漫画卡片 → 进入漫画阅读页
2. 卡片右上角显示菜单按钮（⋮）
3. 菜单包含：编辑、删除

#### 编辑操作
1. 点击编辑 → 打开编辑弹窗
2. 可修改：显示名称、漫画类型、封面上传
3. 保存后更新列表显示

#### 扫描操作
1. 点击"重新扫描" → 调用扫描API
2. 显示加载状态
3. 扫描完成提示新增数量

## 2. 漫画阅读页 `/resources/comics/:id`

### 页面功能
- 显示漫画信息（名称、类型）
- 章节选择器
- 图片浏览（支持条漫/页漫模式）

### 页面布局
```
┌──────────────────────────────────────────────────────────┐
│ [← 返回]  咒术回战  [第1话 ▼]  [条漫|页漫]               │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  ┌────────────────────────────────────────────────────┐  │
│  │                                                    │  │
│  │                    [图片1]                         │  │
│  │                                                    │  │
│  │                    [图片2]                         │  │
│  │                                                    │  │
│  │                    [图片3]                         │  │
│  │                                                    │  │
│  │                    [图片4]                         │  │
│  │                                                    │  │
│  │                         ...                        │  │
│  │                                                    │  │
│  └────────────────────────────────────────────────────┘  │
│                                                          │
│  [← 上一话]                              [下一话 →]      │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 阅读模式

#### 条漫模式（默认）
- 图片垂直排列
- 滚轮滚动浏览
- 适合手机端

#### 页漫模式
- 单张图片居中显示
- 键盘 ← → 翻页
- 空格键下一页
- 支持点击屏幕左右翻页

### 交互说明

#### 章节切换
1. 点击章节下拉 → 显示章节列表
2. 选择章节 → 加载该章节图片
3. 记住用户偏好的阅读模式

#### 图片加载
1. 进入页面时预加载前3张图片
2. 滚动到图片时懒加载
3. 图片错误显示占位图

## 3. 路径设置页 `/resources/settings`

### 页面功能
- 首次使用引导配置
- 管理资源路径
- 测试路径连接

### 页面布局（首次配置）
```
┌──────────────────────────────────────────────────────────┐
│ [面包屑] 资源管理 / 设置                                 │
├──────────────────────────────────────────────────────────┤
│                                                          │
│         ┌─────────────────────────────────────┐         │
│         │                                      │         │
│         │      🖥️ 本地代理服务                 │         │
│         │                                      │         │
│         │   请确保 ProxyService.exe 已运行     │         │
│         │   状态: ● 已连接                     │         │
│         │                                      │         │
│         │   [连接状态检测中...]                │         │
│         │                                      │         │
│         └─────────────────────────────────────┘         │
│                                                          │
│         ┌─────────────────────────────────────┐         │
│         │                                      │         │
│         │      📁 漫画资源路径                 │         │
│         │                                      │         │
│         │   当前路径: D:/MyComics/            │         │
│         │                                      │         │
│         │   [选择文件夹]                      │         │
│         │                                      │         │
│         └─────────────────────────────────────┘         │
│                                                          │
│                    [保存配置]                            │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

### 交互说明

#### 首次配置流程
1. 检测本地代理服务状态
2. 显示连接状态
3. 用户选择漫画资源文件夹
4. 保存后跳转回漫画列表页

#### 文件夹选择
1. 调用系统文件夹选择器
2. 验证路径是否存在
3. 测试路径可访问性

## 5. 移动端适配

### 响应式断点
| 设备 | 宽度 | 卡片列数 |
|------|------|----------|
| 手机 | < 768px | 2列 |
| 平板 | 768px - 1024px | 3列 |
| 桌面 | > 1024px | 4-6列 |

### 移动端交互优化
- 长按漫画卡片显示操作菜单
- 双指缩放图片
- 下拉刷新漫画列表
