# 组件设计要求

## 📦 组件清单

| 组件名 | 用途 | 优先级 |
|--------|------|--------|
| ComicCard | 漫画卡片展示 | P0 |
| ComicEditDialog | 编辑弹窗 | P1 |
| PathSettingDialog | 路径设置弹窗 | P0 |
| ChapterSelector | 章节选择器 | P1 |
| TypeBadge | 类型徽章 | P0 |

---

## 1. ComicCard 漫画卡片

### 设计规格
| 属性 | 值 |
|------|------|
| 宽度 | 自适应栅格 |
| 高度 | 封面3:4 + 信息区 |
| 封面比例 | 3:4 |
| 圆角 | 8px |
| 阴影 | 0 2px 8px rgba(0,0,0,0.08) |
| hover阴影 | 0 4px 12px rgba(0,0,0,0.12) |

### 信息区
| 元素 | 样式 |
|------|------|
| 标题 | 14px, #000000, 500, 单行省略 |
| 副标题 | 12px, #606266 |
| 菜单按钮 | 24x24, 右下角, 悬停显示 |

### 状态设计
| 状态 | 表现 |
|------|------|
| 默认 | 正常显示 |
| 悬停 | 阴影加深，显示菜单按钮 |
| 加载中 | 骨架屏占位 |
| 无封面 | 显示默认占位图 |

### 无封面占位图
按类型显示不同图标和背景色：
| 类型 | 背景色 | 图标 |
|------|--------|------|
| manga | #E3F2FD | 📖 |
| comic | #FFF3E0 | 💬 |
| novel | #E8F5E9 | 📚 |
| picture | #F3E5F5 | 🖼️ |

## 3. ComicEditDialog 编辑弹窗

### 设计规格
| 属性 | 值 |
|------|------|
| 宽度 | 480px (桌面) / 90vw (移动) |
| 圆角 | 12px |
| 内边距 | 24px |

### 布局结构
```
┌─────────────────────────────────────────┐
│ 编辑漫画                            [✕] │
├─────────────────────────────────────────┤
│                                         │
│   ┌─────────────────────────────────┐   │
│   │                                 │   │
│   │         封面预览区域             │   │
│   │         200x267 (3:4)          │   │
│   │         [点击上传图标]          │   │
│   │                                 │   │
│   └─────────────────────────────────┘   │
│                                         │
│   名称:                                  │
│   ┌─────────────────────────────────┐   │
│   │                                 │   │
│   └─────────────────────────────────┘   │
│                                         │
│   类型:                                  │
│   ┌─────────────────────────────────┐   │
│   │ [选择类型                   ▼] │   │
│   └─────────────────────────────────┘   │
│                                         │
├─────────────────────────────────────────┤
│                 [取消]  [保存]          │
└─────────────────────────────────────────┘
```

### 按钮规格
| 按钮 | 背景色 | 文字色 |
|------|--------|--------|
| 取消 | transparent | #606266 |

## 7. 加载状态

### 骨架屏
背景渐变动画：从 #F3F5F7 到 #EEF0F2
动画时长：1.5s

### 加载指示器
使用TDesign Loading组件
尺寸：20px (小) / 32px (中) / 48px (大)
