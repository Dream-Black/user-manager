## 六、Tailwind CSS 配置

```javascript
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        // 品牌色
        primary: {
          DEFAULT: '#0052d9',
          light: '#4d9fff',
          dark: '#003eb8',
        },
        // 功能色
        success: '#00b894',
        warning: '#ffb020',
        danger: '#d93025',
        info: '#0052d9',
        // 分类色
        category: {
          dev: '#0052d9',
          meeting: '#ffb020',
          doc: '#00b894',
          design: '#722ed1',
          debug: '#d93025',
          bug: '#d93025',
        },
        // 优先级
        priority: {
          high: '#d93025',
          medium: '#ffb020',
          low: '#00b894',
        },
      },
    },
  },
}
```

## 八、对比度检查

| 组合 | 对比度 | WCAG AA | WCAG AAA |
|------|--------|---------|----------|
| 深色文字 + 浅色背景 | 12.5:1 | ✅ | ✅ |
| 主色 + 白色 | 4.8:1 | ✅ | ❌ |
| 警告色 + 白色 | 4.6:1 | ✅ | ❌ |
| 危险色 + 白色 | 5.2:1 | ✅ | ❌ |

> 注：文字与背景的对比度需达到 4.5:1 以上才能通过 WCAG AA 标准

## 九、动效中的颜色变化

```
悬停状态：背景色加深 10%
按下状态：背景色加深 15%
禁用状态：整体 opacity 0.5

主题切换：300ms ease-in-out 过渡
```
