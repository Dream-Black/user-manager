**风格定位**: 清新 · 简洁 · 留白 · 轻盈

## 一、设计理念

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   "Less is More" - 少即是多                                          │
│                                                                     │
│   清新：柔和的色彩、轻盈的质感                                        │
│   简洁：减少视觉噪音、突出核心内容                                     │
│   留白：给内容呼吸的空间                                              │
│   轻盈：轻盈的阴影、柔和的圆角                                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 十一、动画规范

```
时长：
• instant: 50ms   - 点击反馈
• fast: 150ms     - 悬停效果
• normal: 250ms   - 弹窗、出现
• slow: 350ms     - 页面切换

缓动：
• ease-out: 出现动画
• ease-in: 消失动画
• ease-in-out: 状态变化
```
